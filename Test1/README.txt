
	This App is a Inventory Manager of Perishable products. The inventory is run through the console. It has a SQLite database. 
	
	When starting the App, the database and the table are created, if it doesn't exist, and the App connects to the database. The Inventory Manager has four options, which are chosen by pressing the keys 1 to 4:
			1. Insert a product. You have to enter all the characteristics of the product (name, expiration date, net price, type, weight and quantity).
			2. List all products. It creates a list of all products inserted into the database.
			3. Search a product. Enter a product name. After search, display and delete the product into the database. 
			4. Exit the app. Exit and close the database.
